🌱 FocusFlow

FocusFlow is a simple and visually engaging focus timer designed to help users stay productive while tracking their study or work sessions.

The app encourages focused work by allowing users to select a plant, set a session duration, and grow their virtual plant by completing focus sessions.

✨ Features
⏱️ Custom focus timer
🌱 Choose a plant before starting a session
📝 Add custom session labels
📖 Session history and records
🌳 Plant grows upon successful completion
💾 Local storage support for saving records
🎨 Modern glassmorphism-inspired UI
📱 Responsive design

🚀 Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
Local Storage API

🎯 How It Works
Open the application.
Select a plant.
Enter a session label.
Choose a focus duration.
Start the timer.
Complete the session to grow your plant.
View completed sessions in the Records section.
